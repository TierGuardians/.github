## 📝 소개
직접 주제를 탐구하고 기획한 개인 금융 관리용 3-Tier 기반 가계부 시스템입니다.
예산 설정, 자산 관리, 소비 내역 기록 등 핵심적인 CRUD 기능을 제공하며,
Spring Security, JWT 인증, 비밀번호 암호화, DB 권한 설정 등으로 보안성 또한 고려하였습니다.
